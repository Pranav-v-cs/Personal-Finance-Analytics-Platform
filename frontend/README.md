# Finlytics Frontend

React SPA for the Finlytics personal finance platform.

## Stack

- React 18 + TypeScript
- Vite + Tailwind CSS
- framer-motion for animations
- Recharts for charts
- @dnd-kit for drag-and-drop dashboard
- OpenRouter AI via backend proxy

## Setup

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm test` | Run tests |
| `npm run lint` | Lint + type-check |
